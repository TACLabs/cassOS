# craignOS

## Introduction

craignOS, anciennement cassOS, nom changé en raison d'un jeu de mots bien plus drôle à mon goût

Système d'exploitation craignos écrit en C++, à vocation éducative

```console
make && qemu-system-i386 -kernel craignOS.bin
```
